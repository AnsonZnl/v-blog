如何让网页只在微信内打开？：

1. 通过userAgent判断

   ```js
   // 判断是否是微信环境
     function getIsWxClient () {
       var ua = navigator.userAgent.toLowerCase();
       if (ua.match(/MicroMessenger/i) == "micromessenger") {
         return true;
       }
       return false;
     };
   ```

   

2. 通过WeixinJSBridge判断(推荐)

   ```js
   // 需要在JS-SDK加载之后判断
   function getIsWxClient(){
    if( typeof WeixinJSBridge !== "undefined" ) {
        return true;
     }else{
         return false
     }
   }
   ```
